To solve this problem, we need to find the increase in the probability \( P \) that would make the chance of successfully typing \( N \) lines equal to the chance of successfully typing \( N-1 \) lines at the original probability.

First, let's understand the probability of success in both scenarios:

1. **Original scenario**: Typing \( N \) lines with probability \( P \):
   \[
   \text{Success Probability}_\text{original} = \left( \frac{P}{100} \right)^N
   \]
   
2. **Modified scenario**: Typing \( N-1 \) lines with probability \( P \):
   \[
   \text{Success Probability}_\text{modified} = \left( \frac{P}{100} \right)^{N-1}
   \]

Our goal is to find the new probability \( P_{\text{new}} \) such that:
\[
\left( \frac{P_{\text{new}}}{100} \right)^N = \left( \frac{P}{100} \right)^{N-1}
\]

Solving for \( P_{\text{new}} \):
\[
\left( \frac{P_{\text{new}}}{100} \right) = \left( \frac{P}{100} \right)^{\frac{N-1}{N}}
\]
\[
P_{\text{new}} = 100 \times \left( \frac{P}{100} \right)^{\frac{N-1}{N}}
\]

The increase in \( P \) is then:
\[
\text{Increase} = P_{\text{new}} - P
\]

Now, let's implement this in Python.

```python
import sys
import math

T = int(sys.stdin.readline())
for case_num in range(1, T + 1):
    N_and_P = sys.stdin.readline().split()
    while len(N_and_P) < 2:
        N_and_P += sys.stdin.readline().split()
    N, P = map(int, N_and_P)
    N = int(N)
    P = float(P)
    ln_P = math.log(P / 100.0)
    exponent = (N - 1) / N
    ln_P_new = exponent * ln_P
    P_new_fraction = math.exp(ln_P_new)
    P_new = P_new_fraction * 100.0
    Increase = P_new - P
    print(f"Case #{case_num}: {Increase}")
```
